## 概述

Stars Api 是一个基于 Python 编写的api集合，它包括一言、天气、温度、甚至是头条新闻。API 中，有些是集成了网络资源 API ，在提到它们时，我们会进行来源的说明。当然，此 API 也会从网页爬取信息，也有自己的一言数据库。
